# 🧮 Calculator

A simple command-line Calculator built using Python. This project performs basic arithmetic operations, validates user input, handles exceptions, and maintains a timestamped history of all calculations using file handling.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧮 Modulus
- 💪 Power
- 📜 View Calculation History
- 🗑️ Clear History
- ⏰ Timestamp for Every Calculation
- ✅ Input Validation
- ⚠️ Exception Handling (Invalid Input & Division by Zero)

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/adityachaurasiya005/Calculator-With-History.git
```

2. Navigate to the project folder

```bash
cd Calculator
```

3. Run the program

```bash
python calculator.py
```
